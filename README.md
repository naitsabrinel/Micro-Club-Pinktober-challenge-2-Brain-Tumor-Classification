# 🧠🎗️ Micro-Club Pinktober: Brain Tumor Classification

Brain tumors are serious and potentially life-threatening conditions that require early and accurate diagnosis. Machine learning can play a crucial role in assisting medical professionals by automating the classification process.

In this challenge, participants are required to build a multi-class classification model to identify the stage of brain tumors.

## 📌 Overview
Task: Multi-class image classification
Classes: Tumor stages 1 to 4
Goal: Predict the correct stage of brain cancer from the given data
## 📂 Dataset

The dataset is available on Kaggle. You can download it using:

kaggle competitions download -c micro-club-pinktober-brain-tumor-classification
## 🎯 Objective

Develop a machine learning model that:

Takes medical data/images as input
Classifies brain tumors into 4 stages (1, 2, 3, 4)
Maximizes performance using the evaluation metric
